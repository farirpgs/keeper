## State Components

### `<TextField>`

A single-line text input field.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <TextField name="name">Character Name</TextField>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the field
- `placeholder`: Placeholder text (optional)
- `children`: Label text for the field
- `tooltip`: Tooltip text (optional)

### `<TextAreaField>`

A multi-line text input field.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <TextAreaField name="description" rows={4}>
    Description
  </TextAreaField>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the field
- `rows`: Number of rows (default: 3)
- `placeholder`: Placeholder text (optional)
- `children`: Label text for the field
- `tooltip`: Tooltip text (optional)

### `<NumberField>`

A numeric input field.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <NumberField name="age" min={0} max={120}>
    Age
  </NumberField>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the field
- `min`: Minimum value (optional)
- `max`: Maximum value (optional)
- `children`: Label text for the field
- `tooltip`: Tooltip text (optional)

### `<SelectField>`

A dropdown selection field.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <SelectField name="class" options={["Fighter", "Wizard", "Rogue"]}>
    Choose Class
  </SelectField>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the field
- `options`: Array of selectable options
- `placeholder`: Placeholder text (optional)
- `defaultValue`: Default selected value (optional)
- `children`: Label text for the field
- `tooltip`: Tooltip text (optional)

### `<CheckboxField>`

A checkbox input field.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <CheckboxField name="myCheckbox" defaultValue={false}>
    Accept terms
  </CheckboxField>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the field
- `defaultValue`: Initial checkbox state (default: false)
- `children`: Label text for the checkbox

### `<Tracker>`

A visual tracker for counting or tracking states.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Tracker name="health" min={1} max={10} asClock={false}>
    Health Points
  </Tracker>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the tracker
- `min`: Minimum number of items (default: 1)
- `max`: Maximum number of items (optional)
- `asClock`: Whether to display as a clock (optional)
- `children`: Label text for the tracker

### `<List>`

Creates a list of items that can be added, removed, and reordered.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <List name="myList" min={1} max={5}>
    <TextField name="item" />
  </List>
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the list
- `min`: Minimum number of items (default: 1)
- `max`: Maximum number of items (optional)
- `children`: The template for list items

### `<RollingTable>`

A table for random rolls with history.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <RollingTable
    name="lootTable"
    label="Loot"
    items={["Gold", "Potion", "Sword"]}
  />
  ```
</MDXCodeBox>

#### Properties

- `name`: Unique identifier for the table
- `items`: Array of possible outcomes
- `groups`: Array of groups with their own items (cannot be used with items)

## UI Components

### `<Heading>`

Displays a bold heading.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Heading>This is a heading</Heading>
  ```
</MDXCodeBox>

#### Properties

- `children`: The heading text

### `<Label>`

Displays a label text.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Label>This is a label</Label>
  ```
</MDXCodeBox>

#### Properties

- `children`: The label text

### `<Detail>`

Displays text with gray styling.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Detail>This is a detail text</Detail>
  ```
</MDXCodeBox>

#### Properties

- `children`: The text content
- `fullWidth`: Whether the text should take full width (optional)

### `<Box>`

A wrapper component that groups content together.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Box>
    <p>Hello, world!</p>
  </Box>
  ```
</MDXCodeBox>

#### Properties

- `children`: The content to be wrapped.

### `<Columns>`

Creates a responsive grid layout with multiple columns.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Columns cols={2}>
    <div>Column 1</div>
    <div>Column 2</div>
  </Columns>
  ```
</MDXCodeBox>

#### Properties

- `cols`: Number of columns (default: 2)
- `children`: The content to be displayed in columns

### `<Divider>`

Creates a horizontal separator line.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Divider />
  ```
</MDXCodeBox>

### `<Row>`

Creates a horizontal flex container.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Row align="center">
    <div>Item 1</div>
    <div>Item 2</div>
  </Row>
  ```
</MDXCodeBox>

#### Properties

- `children`: The content to be displayed in a row
- `align`: Alignment of items ("start", "center", "end", "baseline", "stretch", default: "center")

### `<Stack>`

Creates a vertical flex container with spacing.

<MDXCodeBox client:load language="mdx">
  ```mdx
  <Stack>
    <div>Item 1</div>
    <div>Item 2</div>
  </Stack>
  ```
</MDXCodeBox>

#### Properties

- `children`: The content to be stacked vertically
